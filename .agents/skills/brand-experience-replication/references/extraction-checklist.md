# Extraction Checklist (Sadhaka Canonical Source Pass)

Use this checklist before generating outputs from the skill.

## 1) Brand strategy extraction

- [ ] Mission and worldview captured
- [ ] Audience and problem intensity defined
- [ ] Emotional promise identified
- [ ] Trust posture and conversion philosophy documented

## 2) Design system extraction

- [ ] Semantic color roles defined
- [ ] Typography hierarchy defined
- [ ] Spacing/radius/elevation scales documented
- [ ] Glass/gradient/motion rules captured

## 3) Landing architecture extraction

- [ ] Section order mapped from source
- [ ] Hero structure (claim + CTAs + trust chips) documented
- [ ] Proof modules and objection sequence mapped
- [ ] Final conversion framing documented

## 4) Component pattern extraction

- [ ] Header/navigation behavior captured
- [ ] Hero pattern captured
- [ ] Pillars/education pattern captured
- [ ] Social proof pattern captured
- [ ] Conversion/CTA band pattern captured

## 5) Adaptation readiness

- [ ] Universal principles separated from source-specific details
- [ ] Replicate vs adapt map completed
- [ ] Domain-specific trust translation specified
- [ ] Guardrail and anti-pattern check passed
